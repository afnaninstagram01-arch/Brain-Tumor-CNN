Brain Tumor Classification using CNN

This project uses Convolutional Neural Networks (CNN) to classify brain MRI images into different tumor categories using TensorFlow and Keras.

📌 Project Overview

The model is trained on MRI brain scan images to detect and classify brain tumors. It helps demonstrate how deep learning can be used in medical image analysis.

🛠️ Technologies Used
Python
TensorFlow / Keras
NumPy
Matplotlib
Jupyter Notebook
📂 Dataset

MRI brain images divided into multiple classes such as:

Glioma
Meningioma
Pituitary
No Tumor
🚀 How to Run
Clone the repository
Install required libraries
Run the Jupyter Notebook
🎯 Goal

To build an AI model that can assist in early detection of brain tumors using medical imaging.
